# KeyRecord按键记录工具

## KeyRecord是什么?
这是一款记录计算机上用户一切鼠标键盘操作行为的软件。
所以只要开启了软件，就会在任何环境中实时记录用户操作行为，并生成记录日志。
比如按下了什么按键，点击了什么软件，进入了什么网站等等。
[下载地址](https://gitee.com/Rna_zoro/GlobalHook/releases)

## KeyRecord有哪些功能？

* 记录键鼠输出行为日志
* Alt+A隐藏运行
* 通过图表分析按键行为
* 统计每天按键次数
* 点击和按键热力图查看
