# GlobalHook全局鼠标监测日志工具

## GlobalHook是什么?
是为了监测全局的鼠标键盘状态，而开发出的钩子程序。
也是为了了解什么是钩子。

## GlobalHook有哪些功能？

* 监测键盘和鼠标输出行为日志
* Alt+A隐藏运行
* 自动保存行为日志
* 筛选行为日志
